## Installation 

- `npm install` 
- Go to [config.json](config.json)
- Change `token` to your Bot Token and `ServerID` to your server id the bot is in.
- In the terminal type `node .`

- Setting the bot

- Opening a Ticket

- Closing a Ticket
 - Type `$close <reason>>` (Must Have Staff Role)
